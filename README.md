# my dotfiles

## installation

* `git clone https://github.com/breuerfelix/dotfiles.git ~/dotfiles`
* `bash ~/dotfiles/install.sh`

## content

- distro: arch / manjaro
- window manager: i3
- app launcher: rofi
- terminal: alacritty + tmux
- shell: zsh + oh-my-zsh
- editor: neovim
